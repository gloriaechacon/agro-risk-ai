# PlagaAlert Frontend

This is the frontend application for PlagaAlert, built with React, TypeScript, and Vite.

It provides the user interface for selecting location, choosing crops, and visualizing risk analysis results.

---

## Tech Stack

- React
- TypeScript
- Vite
- CSS (custom styling)

---

## Features

- Interactive map for precise geolocation selection
- Crop selection input
- Risk analysis visualization
- Environmental factor breakdown
- AI-generated insights display
- Risk evolution simulation

---

## API Integration

The frontend communicates with the backend via HTTP requests.

The base URL is defined using:
VITE_API_URL
